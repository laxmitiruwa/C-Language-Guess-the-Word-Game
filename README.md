# ⚙️ Setup Manual

## Requirements

* GCC Compiler
* Code::Blocks, Dev-C++, Turbo C, or Visual Studio Code with C extension

---

## Installation

### Method 1: Clone the Repository

```bash
git clone https://github.com/laxmitiruwa/C-Language-Guess-the-Word-Game.git
```

Open the project folder in your preferred C IDE.

---

### Method 2: Download ZIP

1. Click **Code**
2. Select **Download ZIP**
3. Extract the ZIP file.
4. Open the `.c` file in your preferred IDE.

---

## Compile the Program

Using GCC:

```bash
gcc guess_the_word.c -o guess
```

---

## Run the Program

Windows:

```bash
guess.exe
```

Linux/macOS:

```bash
./guess
```
